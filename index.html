How to Use

  1. Enter commands in the upper text field as comma-separated values
  2. Click Generate UV Buttons
  3. HTML button code will appear in the lower text field

Enter 3 values (separated by commas) to make a toggle-off button. For example,

Uniview Command #1, Uniview Command #2, Button Text

Returns:

<div class="ui-block-b"><button class="uv-toggle-off" data-message-on="Uniview Command #1" data-message-off=" Uniview Command #2"> Button Text</button></div>

Enter 2 values (separated by commas) to make a uniview-message-button. For example,

Uniview Command, Button Text

Returns:

<div class="ui-block-b"> <button class="uv-message-button" data-message="Uniview Command"> Button Text</button> </div>

Try this example list of commands in the upper text field:

pluto.prop.enabled 1, pluto.prop.enabled 0, Pluto On camera.setpivot Earth, Target Earth camera.jumptoEarth, Jump To Earth
